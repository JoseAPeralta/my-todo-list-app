# my-todo-list-app

- A todo list app make from **React js**
- Use **ViteJs** as a build tool
- Use **bootstrap 4.6.0** as a CSS framework

## Install

use `npm install`

## To start dev server

use `npm run dev`

## Building to production

use `npm run build`

## Run test

use `npm run test`
or use `test:watch` to watch mode
